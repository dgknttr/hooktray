# Support

Thanks for using HookTray.

## Questions and Help

For general questions, setup help, and feature discussions, open a GitHub issue using the most relevant template.

Before opening an issue:

- Search existing issues.
- Include your operating system, browser, Node.js version, and .NET SDK version when relevant.
- Include reproduction steps for bugs.

## Security Issues

Do not open public issues for vulnerabilities. Follow [SECURITY.md](SECURITY.md).

## Commercial Support

HookTray is open-source and community-supported. Commercial features, paid plans, and advanced platform capabilities are out of MVP scope unless explicitly approved and documented in the active plan.
